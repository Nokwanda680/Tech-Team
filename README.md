# Student Accommodation (Imported Repo)

Short import of a Django backend and simple front-end files.

Prerequisites
- Python 3.10+ (or the version used by the project)
- Git

Quick start (backend)
```powershell
cd "Student Accomodation - 15 May 2026"\
cd back-end
python -m venv venv
venv\Scripts\Activate.ps1
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

Frontend
- Open `front-end/home page/index.html` in a browser for the static UI preview.
