# Team-8
Strengths
Solves a real and common problem for students (difficulty finding reliable, affordable housing)
Student-focused design makes it more relevant than generic listing platforms
Includes an affordability checker, adding practical decision-making value
Simple, clear structure (listings + filters + details) makes it user-friendly
Addresses safety concerns through verification indicators
Manageable scope (e.g. 5 listings) makes it realistic to build within timeline
Weaknesses
Limited number of listings (demo data only, not a full marketplace)
No real-time data or landlord integration
Trust system (verification) may be simulated rather than fully implemented
May lack advanced features like messaging or live updates
Opportunities
Expand to include real landlords and live listings
Add student reviews/ratings for accommodations
Partner with universities or student housing offices
Introduce roommate matching or shared housing features
Strengthen legal/educational content:
tenant rights
lease guidance
Scale to multiple campuses or cities
Threats
Competition from existing platforms and social media groups
Risk of fraudulent listings or scams in real-world expansion
Data privacy concerns when handling user information
Low adoption if students stick to other platforms (e.g. WhatsApp/Facebook groups)
Technical limitations if the platform tries to scale too quickly
